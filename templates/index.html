Пошаговое руководство:

1

Подготовка файлов

Подготовьте два файла Excel:

  • Файл из системы 1С - содержит данные о товарах и остатках
  • Файл из системы Synteka - содержит реестр операций

Файлы должны быть в формате .xlsx или .xls

2

Загрузка файлов на сервер

В разделе "Загрузка исходных файлов":

  • Нажмите "Выберите файл" для файла из 1С
  • Выберите файл и нажмите "Загрузить"
  • Повторите операцию для файла из Synteka

Дождитесь сообщения об успешной загрузке каждого файла

3

Проверка загруженных файлов

В разделе "Состояние файлов на сервере" убедитесь, что:

  • Переключатель "Файл из 1С" стал зеленым
  • Переключатель "Файл из Synteka" стал зеленым

Если переключатели серые - файлы не загружены

4

Запуск обработки

После успешной загрузки обоих файлов:

  • Нажмите кнопку "Запустить обработку"
  • Следите за сообщениями в консоли статуса
  • Процесс может занять от 30 секунд до 2 минут

Не закрывайте страницу во время обработки

5

Получение результата

После завершения обработки:

  • Переключатель "Файл результат" станет зеленым
  • Кнопка "Скачать результат" станет активной
  • Нажмите кнопку для скачивания готового файла

Результат содержит объединенные и сопоставленные данные

!

Важные замечания

  • Используйте кнопку "Очистить файлы" для удаления старых файлов
  • При ошибках используйте кнопку "Показать логи" для диагностики
  • Сессия работы длится 24 часа
  • Рекомендуется использовать файлы размером не более 50 МБ
